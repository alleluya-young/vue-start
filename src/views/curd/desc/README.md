## ProCurdForm

- 根据 Curd 中标记的 columns 生成的 Desc

## API

继承 ProDesc 中的所有属性

| 名称       | 说明                                                                 | 类型     | 默认值        |
| ---------- | -------------------------------------------------------------------- | -------- | ------------- |
| `clsName`  | class 名称                                                           | `string` | pro-curd-desc |
| `signName` | column 中标记的属性名，不配置的话默认会把 columns 都会加入到 Form 中 | `string` | --            |
